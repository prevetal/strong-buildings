Основные конструктивные
элементы

Каркасная система — основными несущими конструкциями каркасов серии Стронг являются сплошностенчатые рамы из сварных двутавров;

Устойчивость и геометрическая неизменяемость здания обеспечивается:

  • в поперечном направлении — конструкциями несущих рам и связями торцевых фахверков;
  • в продольном направлении — вертикальными связями по каждому ряду колонн и горизонтальными связями по покрытию;
Прогоны покрытия, торцевые балки — выполняются из гнутых швеллеров;
Стойки фахверка — выполняются из гнутых швеллеров.
Сопряжение колонн с фундаментами жёсткое, с ригелем — шарнирное;
Коньковый узел сопряжения отправочных элементов ригеля — фланцевый на высокопрочных болтах с предварительным натяжением;
Крепёж на обычных и высокопрочных болтах.

Статическая модель стального каркаса представляет собой портальную раму с шарнирным сопряжением колонн с фундаментом. Двухпролетный каркас — двойная портальная рама с аналогичным сопряжением колонн с фундаментом. Бесшарнирное соединение применяется только в особых условиях, например, для высоких зданий или при строительстве в сейсмически опасных зонах. Возможна организация опор для мезонинов. Несущая плита междуэтажного перекрытия устанавливается на отдельных опорах.
